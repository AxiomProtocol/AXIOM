# SWF Smart Contract Addresses

## Core SWF Ecosystem Contracts

### SWF Main Token Contract
- **Address:** 0x83E17aeB148d9b4b7Be0Be7C87dd73531a5a5738
- **Network:** BSC Mainnet
- **Type:** Main SWF utility token with real estate tokenization features
- **Verified:** June 14, 2025 on BSCScan
- **Features:** Built-in property tokenization, Kinesis gold backing, Merkle tree airdrops

### SWFMaster DAO Contract
- **Address:** 0x1680F203Aafcbc64adAbD54C15472F7283aA8d08
- **Network:** BSC Mainnet
- **Type:** Governance and treasury management contract
- **Features:** Proposal creation, voting mechanisms, treasury controls, daily transfer limits

### SWFMasterDAO Ultimate Contract
- **Address:** 0xf1aFbA4804dB581f5eC43f7AFC3e522728ddDcF0
- **Network:** BSC Mainnet
- **Type:** Advanced DeFi vault with staking pools and governance
- **Features:** Multi-pool staking, auto-staking, commodity backing, ICP Oracle integration, ISO20022 compliance

### Sovran Launchpad Contract
- **Address:** 0x0C0bfd4E4170411bc9665A368FaFaB3048883C4C
- **Network:** BSC Mainnet
- **Type:** Multi-token contribution platform for fundraising
- **Verified:** July 2, 2025 on BSCScan
- **Features:** Whitelist management, multi-token support (WBNB, WETH, WXRP, WBTC), contribution limits, treasury management

## Property Tokenization System
- **Implementation:** Integrated into main SWF token contract
- **Real Estate Assets:** Managed through tokenizeRealEstate() function
- **Backing System:** Commodity and reserve backing through Ultimate DAO contract
