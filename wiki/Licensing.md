# Commercial Licensing

## Overview

Axiom Protocol technology is proprietary and protected by copyright, patents (pending), and trade secrets. All commercial use requires a paid license.

---

## License Types

### 1. Full Platform License
Complete access to deploy Axiom Protocol technology.

**Includes:**
- All smart contracts
- Frontend applications
- Backend services
- Database schemas
- API access

**Use Cases:**
- White-label smart city deployment
- Full ecosystem licensing
- Government/municipal partnerships

**Pricing:** Contact for enterprise quote

### 2. Smart Contract License
License specific smart contracts for integration.

**Available Contracts:**
- Property tokenization (ERC-1155)
- Governance system
- Staking infrastructure
- DePIN node management
- Banking primitives

**Pricing:** Per-contract licensing, starting at $25,000

**Requirements:**

- Accredited investor status

The following are **NOT** permitted without explicit license:

- Commercial deployment
- Revenue-generating applications
- Competing product development
- White-label resale
- API commercialization

- Code modification/derivatives
- Integration into third-party products
